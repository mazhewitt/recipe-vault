## MODIFIED Requirements

### Requirement: Navigation arrows use touch-friendly sizing

The navigation arrow buttons SHALL increase in size on touch device viewports to meet minimum tap target requirements.

#### Scenario: Arrow size on desktop

- **WHEN** the viewport is wider than 1024px
- **THEN** the navigation arrows are 29px x 29px

#### Scenario: Arrow size on tablet and mobile

- **WHEN** the viewport is 1024px or less
- **THEN** the navigation arrows are at least 44px x 44px
